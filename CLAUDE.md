# toggle-docs

Internal documentation hub for Toggle Solutions. Docusaurus 3.x site that auto-syncs all content from toggle-brain on every push to main.

## What this repo is

This repo is intentionally thin — it contains only Docusaurus config and a sync workflow. **Never edit `docs/` content directly.** All content lives in toggle-brain (`~/Desktop/toggle-brain`). The `docs/` folder is fully replaced on every sync run.

## Relationship to toggle-brain

```
toggle-brain (source of truth)
  └── .github/workflows/notify-docs.yml
        fires repository_dispatch → toggle-docs on every push to main

toggle-docs (this repo)
  └── .github/workflows/sync-content.yml
        sparse-checks out toggle-brain → rsyncs 7 zones into docs/ → commits → pushes
        Cloudflare Pages auto-deploys on every push
```

Content zones synced from toggle-brain:
- `docs/brain/` — Knowledge Base (services, pricing, voice, case studies, team, verticals)
- `docs/playbooks/` — Step-by-step runbooks
- `docs/generators/` — Slash-command recipes
- `docs/prompts/` — Prompt library
- `docs/templates/` — Empty shells for client work
- `docs/clients/` — Per-client briefs, strategy, creative, reports
- `docs/cockpit/` — Daily focus, journal, decisions, todos

## Running locally

```bash
npm start          # dev server at http://localhost:3000
npm run build      # production build → build/
npm run serve      # serve production build locally
```

## Key files

| File | Purpose |
|---|---|
| `docusaurus.config.ts` | Site title, URL, plugins, navbar, footer |
| `sidebars.ts` | Autogenerated sidebar from docs/ folder structure |
| `src/css/custom.css` | Toggle brand color overrides |
| `docs/intro.md` | Welcome page (the only manually maintained doc) |
| `.github/workflows/sync-content.yml` | Sync workflow — triggered by repository_dispatch from toggle-brain |

## GitHub Actions secrets required

| Secret | Purpose |
|---|---|
| `BRAIN_READ_TOKEN` | PAT with `contents:read` on toggle-brain — used by sync workflow to sparse-checkout content |

Add at: `https://github.com/toggle-workspace/toggle-docs/settings/secrets/actions`

## Hosting

- **URL:** `https://brain.toggle.solutions`
- **Platform:** Cloudflare Pages (auto-deploys on push to `main`)
- **Auth:** Cloudflare Access — Google SSO restricted to `@toggle.solutions` accounts
- **Build command:** `npm run build`
- **Output directory:** `build`
